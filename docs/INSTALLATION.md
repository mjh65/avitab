# Installation of the Avitab plugin for X-Plane

After downloading the Avitab package and extracting, you should find these installation instructions in a file at the top-level alongside a directory `Avitab`.

To install the Avitab plug-in for X-Plane, first locate your X-Plane installation. The top-level directory (which contains the X-Plane executable) will contain a sub-directory `Resources` and inside this is a sub-directory `plugins`. Open this directory in your file explorer.

## First-time Installation

The `Avitab` directory in the downloaded package should be copied into the X-Plane `plugins` directory.

## Update of Existing Installation

If there is already an `Avitab` directory in the `plugins` directory then there is an existing installation of `Avitab`. We recommend that you make a backup copy of this directory somewhere outside of the X-Plane directory tree. You may wish to create a completely fresh installation by **moving** the entire directory to a backup area, however, any settings will then revert to the defaults.

The `Avitab` directory in the downloaded package should be copied into the X-Plane `plugins` directory. If there is a previous installation of `Avitab` **do not** copy the new `Avitab` **into** the previous one, but rather *merge* the new `Avitab` into the previous one so that any of your working files remain untouched, but the plugin binary will be updated.

> At a minimum, you can replace only the file `Avitab.xpl` for your platform since the other files are normally unchanged.
