# Installing the Avitab Pre-Release Test Assets

After downloading the Avitab pre-release test assets package and unpacking, you should find these installation instructions in a file at the top-level alongside directories `Output` and `Resources`.

These two directories contain additional files to be added to the X-Plane installation so that they are available in the expected locations when the pre-release tests are carried out.

To install the Avitab pre-release test assets, first locate your X-Plane installation. The top-level directory contains the X-Plane executable and a number of directories and files, at least including `Output` and `Resources`.

Typically if you use a file manager to copy the two directories `Output` and `Resources` from the test assest package into the X-Plane top-level directory the contents will be added to (merged with) files that are already present in the directory hierarchy.

If your file manager wants to replace the existing directories, then STOP! You will need to copy the files individually into the correct locations.
